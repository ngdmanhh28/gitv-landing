# Landing website for [Gitv](https://github.com/V-Bach/bettergit)
A lightweight, high-performance documentation system built for clean, distraction-free reading. It features dynamic content loading, smart sidebar navigation, and a seamless user experience.

# Key Features
Dynamic Content Loading: Switch between documentation topics instantly without reloading the page using efficient DOM manipulation.

Smart Sidebar Navigation: Automatically syncs the "active" state and handles nested group expansion.

Optimal Readability: Standardized styling for headings, code blocks, and lists to ensure a professional look.

Auto-Reset Scrolling: Automatically scrolls to the top of the content area whenever a new article is selected, ensuring a consistent user experience.

Internal Navigation: Seamlessly handles internal links to jump between topics while maintaining state synchronization.

# Tech Stack
HTML: Semantic document structure.

CSS: Flexbox layout for sidebar and content management.

JavaScript: Logic for dynamic article switching and scroll management. No dependencies required.
